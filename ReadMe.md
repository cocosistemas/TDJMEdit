# Componente TDJMEdit

Un TEdit que sirve para:
- strings
- fechas
- números
- tiempo (hh:mm)

Componentes similares hay muchos pero este tiene varias particularidades útiles, espero que les sea de utilidad.

edittype..

esc

uso sin \ y sin :

ctrl-a ctrl-c ctrl-v

currencysymbol, decimales

validacion onchange y cancelacion con undo (como cosa interna a destacar)

en caso de error

completado y formateado de nuemeros

f2 y f3 en fechas

focuscolor colorfocus

etc
